# 📚 Banco de Dados

Este repositório foi criado para armazenar todos os materiais, exercícios, roteiros de práticas e projetos desenvolvidos durante a disciplina de **Banco de Dados** do curso de `<Ciência da Computação>` na `<Universidade federal de Campina Grande >`.

## 🎯 Objetivo

O objetivo principal deste repositório é documentar o meu aprendizado e evolução na modelagem, criação e manipulação de bancos de dados relacionais (e/ou não relacionais).

## 🗂️ Estrutura do Repositório

O repositório está organizado da seguinte forma:

* 📂 **`/exercicios`**: Resoluções de listas de exercícios de fixação.
* 📂 **`/roteiros`**: Scripts e consultas SQL desenvolvidos durante as aulas práticas e laboratórios.
* 📂 **`/projetos`**: Trabalhos maiores ou projetos finais da disciplina.
* 📂 **`/anotacoes`**: Resumos, mapas mentais ou anotações teóricas sobre a matéria.
